# Getting Started

This guide covers the supported `0.3.0` local-first setup for ZoteroCopilot.

## 1. Install the Python package

```bash
pip install zotero-mcp-server
```

or:

```bash
uv tool install zotero-mcp-server
```

## 2. Build the Zotero plugin

```bash
python3 packaging/plugin/build_xpi.py
```

This produces:

- `dist/plugins/zotero_copilot_0.3.0_zotero7_plugin.xpi`
- `dist/plugins/zotero_copilot_0.3.0_zotero8_plugin.xpi`

Install the XPI that matches your Zotero major version.

## 3. Build the helper package

- macOS: follow [../packaging/macos/README.md](../packaging/macos/README.md)
- Windows: follow [../packaging/windows/README.md](../packaging/windows/README.md)

The public release outputs are:

- `dist/releases/zotero_copilot_0.3.0_helper_macos_arm64.tar.gz`
- `dist/releases/zotero_copilot_0.3.0_helper_windows_x64.zip`

## 4. Extract the helper archive

Extract the archive and keep the whole extracted helper directory intact.

- macOS executable:
  - `zotero_copilot_0.3.0_helper_macos_arm64/zotero_copilot_0.3.0_helper_macos_arm64`
- Windows executable:
  - `zotero_copilot_0.3.0_helper_windows_x64/zotero_copilot_0.3.0_helper_windows_x64.exe`

Do not move only the executable. The `_internal/` directory must remain next to it.

## 5. Configure the plugin

Open Zotero and then open the Zotero Copilot preferences page. Configure:

- the helper executable path
- the buffer directory
- whether write operations are enabled
- the local MCP port and token if you need custom values

The helper now follows the Zotero lifecycle automatically:

- Zotero startup: helper is ensured automatically
- port or token change: helper restarts automatically
- Zotero shutdown: plugin-managed helper stops automatically

## 6. Connect your MCP client

The helper-facing MCP URL is:

```text
http://127.0.0.1:8000/mcp
```

The helper-facing desktop bridge URL is:

```text
http://127.0.0.1:8000/zero-mcp
```

Use the configuration snippet generated by the plugin UI whenever possible.

## 7. macOS quarantine note

If macOS blocks the helper after extraction:

```bash
xattr -dr com.apple.quarantine /path/to/extracted/zotero_copilot_0.3.0_helper_macos_arm64
```

Then re-select the helper executable in Zotero preferences and test the connection again.
