# Getting Started

This guide assumes ZoteroCopilot is already installed from the release packages. If you still need to build it yourself, use [development.md](development.md).

## First-Time Plugin Setup

Open Zotero Copilot preferences and check these items:

- **Helper executable path**
  Choose the executable inside the extracted helper directory.
- **Buffer directory**
  PDFs are staged here before import into Zotero.
- **Allow writes to Zotero**
  Turn this on only if your MCP client should create, move, import, or delete items.
- **Port and token**
  Leave the defaults unless you have a specific reason to change them.

The plugin manages the helper automatically:

- Zotero startup: helper is started or recovered automatically
- Port or token change: helper restarts automatically
- Zotero shutdown: the plugin-managed helper stops automatically

## Connect Your MCP Client

The default MCP endpoint is:

```text
http://127.0.0.1:8000/mcp
```

The helper-facing bridge proxy is:

```text
http://127.0.0.1:8000/zero-mcp
```

Use the configuration snippet generated by the plugin UI whenever possible. If you change the port or token later, copy the new client configuration again.

## Common Problems

### The helper does not start

Check these items first:

- The selected path points to the executable, not just the extracted folder
- The full helper directory is still intact
- `_internal/` still sits next to the executable

### The connection test fails

Check:

- Zotero is still running
- The helper path is correct
- The selected port is not already in use
- Your MCP client is using the latest copied configuration

### PDF import fails immediately

Check:

- The buffer directory exists and is writable
- The file is a real PDF
- The write toggle is enabled if your client is trying to import into Zotero

### macOS blocks the helper

Run:

```bash
xattr -dr com.apple.quarantine /path/to/extracted/zotero_copilot_0.3.0_helper_macos_arm64
```

Then re-select the executable in Zotero preferences and test the connection again.
